# 贡献者公约 / Contributor Covenant Code of Conduct

## 承诺

为营造开放友好的社区,作为贡献者和维护者,我们承诺使参与本项目的每个人都不受骚扰,
无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、
社会经济地位、国籍、个人外貌、种族、宗教或性取向。

## 标准

**鼓励**的行为:用友善和尊重的方式交流;理解不同意见;接受建设性反馈;
关注社区整体利益。

**不可接受**的行为:性化语言或图像、钓鱼/人身攻击、公开或私下骚扰、
未经明确许可泄露他人私有信息、其他不专业的行为。

## 执行

维护者有权对违反本公约的行为予以警告、删除评论或封禁。
违规可私密报告至维护者(见 README 联系方式)。

本公约改编自 [Contributor Covenant][homepage] 2.1。
